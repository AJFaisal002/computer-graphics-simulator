# 🖥️ Computer Graphics Algorithms Simulator

An interactive Computer Graphics Algorithms Simulator built with **Streamlit** and **Matplotlib**.

🔗 Live App: https://graphics02.streamlit.app/

## 📌 Overview

This simulator demonstrates fundamental Computer Graphics algorithms in both **2D** and **3D**.

It is designed for:
- Academic learning
- Visual understanding of algorithms
- Exam preparation
- Interactive experimentation


## 🧠 Educational Purpose

This project demonstrates:

- Rasterization techniques
- Depth sorting
- Visibility determination
- Projection transformations
- Filling algorithms

It is suitable for:
- Computer Graphics courses
- Undergraduate projects
- Practical lab demonstrations

---
## 🚀 Implemented Algorithms

### 🔹 2D Algorithms

- ✅ DDA Line Drawing
- ✅ Bresenham Line Drawing
- ✅ Midpoint Circle Algorithm
- ✅ Scan-Line Polygon Fill
- ✅ Boundary Fill
- ✅ Flood Fill

### 🔹 2D Transformations
- ✅ Translation  
- ✅ Scaling  
- ✅ Rotation  
- ✅ Shearing  
- ✅ Reflection 

### 🔹 3D Algorithms

- ✅ Back-Face Culling
- ✅ Painter’s Algorithm
- ✅ Z-Buffer Algorithm
- ✅ Perspective Projection
- ✅ Depth-based Shading
